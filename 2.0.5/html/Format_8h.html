Description détaillée

Définition des namespaces Compression, SampleFormat, Photometric, ExtraSample et Rok4Format.

  • SampleFormat : gère les types de canaux acceptés par les classes d'Image
  • Compression : énumère et manipule les différentes compressions
  • Rok4Format : énumère et manipule les différents format de données ROK4
  • Photometric : énumère et manipule les différentes photométries
  • ExtraSample : énumère et manipule les différents type de canal supplémentaire
