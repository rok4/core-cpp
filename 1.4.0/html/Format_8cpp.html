Implémentation des namespaces Compression, SampleFormat, Photometric, ExtraSample et Format.

  • Format : énumère et manipule les différentes format d'image
  • Photometric : énumère et manipule les différentes photométries
  • ExtraSample : énumère et manipule les différents type de canal supplémentaire
  • ContextType : gère les types de contextes pour le stockage
