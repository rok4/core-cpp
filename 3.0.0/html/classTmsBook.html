TileMatrixSet * TmsBook::get_tms ( std::string  id)
static

Retourne le TMS d'après son identifiant.

Si le TMS demandé n'est pas encore dans l'annuaire, ou que l'on ne veut pas de cache, il est recherché dans le répertoire connu et chargé

Paramètres
[in]idIdentifiant du TMS voulu

Return the TMS according to its identifier

If TMS is still not in the book, or cache is disabled, it is searched in the known directory and loaded

Paramètres
[in]idWanted TMS identifier
